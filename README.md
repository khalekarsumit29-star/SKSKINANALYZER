# 🌿 SK Skin Analyzer

An **AI-powered Skin Analysis Web Application** that analyzes facial skin conditions using image processing and AI models and provides personalized skincare insights and product recommendations.

Built using **React + TypeScript + Express + TensorFlow + Gemini AI**, this project allows users to upload or capture facial images and receive automated skin health analysis.

---

## 🚀 Project Overview

**SK Skin Analyzer** is a full-stack web application designed to:

- Detect facial skin characteristics from images
- Analyze skin conditions using AI
- Store user analysis history
- Recommend skincare products
- Provide a dashboard for tracking skin health progress

The system integrates **computer vision**, **AI APIs**, and **database storage** to create an intelligent skincare assistant.

---

## ✨ Features

- ✅ AI-based skin condition analysis
- ✅ Facial image upload or webcam capture
- ✅ Acne, dryness, oiliness & pigmentation scoring
- ✅ Personalized skincare recommendations
- ✅ Analysis history tracking
- ✅ Product recommendation system
- ✅ Interactive dashboard visualization
- ✅ Responsive modern UI
- ✅ Local SQLite database storage
